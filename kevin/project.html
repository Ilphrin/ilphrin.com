TuxleTriad

Tuxle Triad est un jeu de cartes inspiré de Triple Triad, un mini-jeu provenant de Final Fantasy VIII. Dans ce jeu deux joueurs s'affrontent en posant des cartes possédant 4 valeurs permettant d'attaquer les autres cartes autour de celle-ci. Le but est d'avoir le plus de cartes de sa couleur lorsque toutes les cases du terrain ont été remplies.

Ce jeu à été développé en Python, et utilise la bibliothèque graphique Pygame. Ce jeu implémente Gettext pour gérer la traduction dans d'autres langues (le fichier .pot est disponible sur le dépot Github). Tuxle Triad est constamment en développement et disponible sous licence MIT, vous pouvez récupérer le code source sur Github (cliquer sur le titre pour le lien). Si vous avez des suggestions d'amélioration, voire si vous avez des compétences en Python n'hésitez pas. Si vous maitrisez une langue dans laquelle Tuxle Triad n'est pas traduit vous pouvez proposer votre fichier de traduction!

Voici la liste des personnes qui ont travaillé sur ce jeu:

  • Kevin Pellet (Ilphrin): Développement
  • Yunero Kisapsodos: Graphismes
  • Devyn Collier Johnson: Création et hébergement du paquet .deb

Pour l'installation il y a deux possibilités. Si vous êtes sous Linux sur une base Ubuntu vous pouvez télécharger le paquet sur le dépot de Devyn C. Johnson a l'adresse suivante: https://launchpad.net/~devyncjohnson-d/+related-packages. Soit vous pouvez cliquer le lien du dépot Github et télécharger les sources dans une archive .zip.

Le jeu est, pour le moment, disponible dans les langues suivantes: Français, Anglais

Si vous souhaitez rajouter votre traduction, il vous suffit d'aller sur la page Github du projet, et de créer votre fichier de traduction gettext en format .po à partir du fichier tuxle-triad.pot dans le dossier translations. (Pour plus d'informations, n'hésitez pas à consulter la documentation de Gettext).

Dodgell

Il faut savoir que j'adore le Web, et j'adore les jeux vidéos. Il y a peu de temps j'ai découvert Superpowers et pour apprendre à l'utilser je me suis dit que le mieux était de bosser sur un petit jeu pour démarrer, et l'améliorer jusqu'à passer à un autre jeu. Dodgell est donc mon tout premier jeu web avec Superpowers (tout premier jeu web tout court d'ailleurs).

A l'écran vous avez des ennemis qui tombent du ciel, et vou pouvez bouger un bonhomme en bas avec les flèches. Votre but est d'esquiver les ennemis le plus longtemps possible, faisant grimper votre score. Vous pouvez y jouer sur ma page de projet itch.io, soit avec CE LIEN, soit en cliquant sur l'image du jeu.
